# Reflexes

Application Android qui est un petit jeu de reflexes.  
- Mode 1 joueur : Consiste à appuyer le plus vite possible sur les 5 boutons qui vont apparaitre aléatoirement. Gestion des high scores implémentées
- Mode 2 joueurs : 2 boutons apparaissent sur chaque moitié de l'écran, le plus rapide des 2 à appuyer sur son bouton marque un point. Match en 5 rounds
